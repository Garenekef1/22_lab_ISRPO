Проект содержит два модуля:
calculator - калькулятор с обработкой событий и валидацией ввода
tic-tac-toe - заготовка под приложение крестики-нолики

Функциональность калькулятора:
поддерживаются операции +, -, *, /
поддерживается ввод десятичных чисел
есть очистка выражения через кнопку C
есть вычисление через кнопку =
при некорректном выражении показывается Ошибка
запрещаются двойные операторы, дублирующиеся точки и выражения с оператором в конце

Ограничения:
не поддерживаются скобки
выражение вводится только кнопками интерфейса

Обработка ошибок:
ошибочное выражение переводит калькулятор в состояние ошибки
после ошибки следующее нажатие очищает состояние
кнопка C всегда полностью сбрасывает выражение

Архитектура:
calculator/index.html - разметка калькулятора
calculator/style.css - стили
calculator/calculator.js - класс Calculator с основной логикой
calculator/main.js - инициализация калькулятора
tic-tac-toe/index.html, style.css, main.js - отдельный модуль проекта

Схема взаимодействия компонентов:
кнопка в index.html -> handleButtonClick в Calculator -> updateDisplay в display

Логика работы:
при нажатии кнопки значение передается в обработчик
если это C - выполняется clear
если это = - выполняется calculate с проверкой выражения
если это число или оператор - вызывается addToExpression с валидацией
после каждого действия обновляется display
папка img используется для скриншотов по шагам лабораторной
